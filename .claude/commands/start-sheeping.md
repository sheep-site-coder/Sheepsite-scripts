# Start Sheeping

Get ready to work on the Sheepsite-scripts project.

## Instructions

1. Pull the latest code from git:
   ```bash
   git pull
   ```

2. Read the conversation snapshot to understand the current project state:
   - Read `docs/conversation-snapshot.md`

3. Read the buildings config to know what communities are set up:
   - Read `buildings.php`

4. Summarize for me:
   - Current project status
   - What communities (buildings) are configured
   - Any recent changes from the git pull
   - Any open issues or next steps from the snapshot

5. Ask me what I'd like to work on today.
